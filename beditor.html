beditor

Options

--install-completion

Install completion for the current shell.

--show-completion

Show completion for the current shell, to copy it or customize the installation.

clip-bathy

Clip the bathymetry of MITgcm between min and max values.

beditor clip-bathy [OPTIONS] BATHY_FILE

Options

--mindepth <mindepth>

Minimum depth (-ve downwards), z[z > mindepth]=landvalue

Default:

-5

--maxdepth <maxdepth>

Maximum depth (-ve downwards), z[z < maxdepth]= maxdepth

--landvalue <landvalue>

Land value

Default:

100.0

Arguments

BATHY_FILE

Required argument

MITgcm bathymetry file
